Welcome to the Toggl to OpenAir timesheet extension. Please set the information below and click create!

API key is found within your Toggl profile.

The start date is the date timesheet entries are pulled from Toggl. An end date is automatically set as six days from the start date to pull a full week.

Just getting started? Open a completed timesheet and simply click Import Projects to import projects into Toggl. Need more help? Click Help!
